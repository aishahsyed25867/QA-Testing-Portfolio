# QA & API Testing Portfolio

This repository contains practical Quality Assurance artifacts demonstrating manual test case design, edge-case analysis, and automated API validation.

## 📁 Repository Structure
* `TEST_CASES.md`: Manual test suite covering positive, negative, security, and boundary scenarios for web authentication.
* `API_Tests.postman_collection.json`: Postman test collection validating REST API endpoints, HTTP status codes, and JSON schema payloads.

## 🚀 How to Run the Postman Collection
1. Import `API_Tests.postman_collection.json` into Postman.
2. Execute the collection via Postman Collection Runner.
